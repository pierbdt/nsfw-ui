Buttons

Use .btn as the base, then add a variant.

Variants

<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-danger">Danger</button>

<button class="btn btn-outline">Outline</button>

Theme

Override CSS variables in your project's app.css to customize.

Default Variables

@theme {
  --color-primary: #4f46e5;
  --color-primary-content: #ffffff;
  --color-primary-focus: #4338ca;
  --color-secondary: #6b7280;
  --color-danger: #dc2626;
  --color-success: #16a34a;
  --color-warning: #f59e0b;
  --color-info: #3b82f6;
  --radius-btn: 0.375rem;
  --radius-input: 0.375rem;
  --radius-card: 0.5rem;
  --radius-badge: 9999px;
}

Override Example

/* resources/css/app.css */
@import "tailwindcss";
@import "nsfw-ui";

@theme {
  --color-primary: #e11d48;      /* rose */
  --color-primary-focus: #be123c;
  --radius-btn: 9999px;          /* pill buttons */
}
